Typeface Selection

Two typefaces form the typographic system: Cormorant Garamond for display headings and DM Sans for all body and UI text. Both are loaded via Google Fonts and applied consistently across every page.

Aa Bb Cc
Cormorant Garamond Display headings · H1, H2, H3, pull quotes Weights used: Light 300, Regular 400, SemiBold 600, Italic variants Carries warmth, history, and cultural depth. The serif structure evokes printed heritage — fitting for a storytelling platform. The italic variant adds expressiveness to pull quotes and accent text.
Aa Bb Cc
DM Sans Body text · UI labels · navigation · buttons Weights used: Light 300, Regular 400, Medium 500 Humanist sans-serif designed for screen legibility. Slightly rounded letterforms feel approachable without being childish. Excellent legibility at small sizes — critical for accessibility (DR2, F2).

Type Scale

The type scale uses clamp() for responsive fluid sizing between mobile and desktop breakpoints. This eliminates the need for multiple media-query overrides and ensures comfortable reading at any viewport width.

Accessibility rationale → DR2, F2

Body text minimum 18px exceeds WCAG 1.4.4 requirements. Line height 1.8 aids readers with dyslexia or low vision. Maximum line length 68 characters prevents eye fatigue on wide screens. All rationale traceable to Sunita's interview (F2) and DR2 (WCAG 2.1 AA compliance).
